== Description ==

This plugin allows you to display advertisements from various different networks or ad sizes!

You can have 3 different widgets with ad code on, so you can quickly select what code you wish to display, and then put them in the position that YOU want!

There are options to place code before and after your posts - This way, you can maximise your revenue without doing anything extra!



It's very easy to setup the plugin - Simply enable it, alter the settings and if you want, enable the widget(s)!

== Installation ==

1. Upload `jr-ads.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Ads Settings in your Administration Panel
4. Optional: Enable the Widget(s) through the Appearance --> Widgets tab.
